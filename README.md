# Spora
Spora is an interactive scientific website, focus on the amazing world of fungi and the incredible capacities that they have, it is divided in three different pages engineering, medicine and sustainability, each one explains the role fungi could play in the future in each field.

You can try out the website through this link https://isabellagg19.github.io/Spora/

I made it using html, css, and to give it the interactive touch java script

This is an amazing website to learn something new, and start paying more attention to these little organisms that despite their small appearance they can make a huge impact in our world.
I hope you can check it out, and enjoy exploring it as much as I enjoyed making it
